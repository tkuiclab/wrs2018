# TIMDA IBM Assistant Web UI

## Build-up
Please prepare Node.js, npm, and rosbridge server

### Installization
```bash
$ cd <timda_ws>/src/web/myapp
$ npm install
```
### Run Web Server
```bash
$ node index.js
```
### Run Assistant Strategy
```bash
$ rosrun strategy custom_server.py
```
